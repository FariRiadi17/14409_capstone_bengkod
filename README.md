# Capstone Project

## Project Overview
This is a capstone project for Bengkel Koding Data Science course at Universitas Dian Nuswantoro. The project predicts customer churn in telecommunications using machine learning.

## Files Included
1. `best_churn_model.pkl` - Trained machine learning model
2. `streamlit_app.py` - Streamlit web application
3. `requirements.txt` - Python dependencies
4. `feature_names.pkl` - Feature names for the model
5. `all_model_results.pkl` - Model evaluation results
6. Various `.png` files - Visualizations and charts

## How to Run Locally
1. Install requirements: `pip install -r requirements.txt`
2. Run Streamlit app: `streamlit run streamlit_app.py`

## How to Deploy on Streamlit Cloud
1. Create GitHub repository with all files
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub repository
4. Set main file to `streamlit_app.py`
5. Deploy!

## Model Performance
- Best Model: Voting Classifier (Tuned)
- F1-Score: 0.62
- Accuracy: 0.81
- Precision: 0.67
- Recall: 0.58
